API overview

FRAME Virtual Fiscalisation provides REST APIs to manage devices, issue fiscal receipts, and query compliance status. Use the API hub to explore endpoints, authentication, and sample requests.

Getting started with the API

Sign in to the FRAME Virtual Fiscalisation console to generate API keys. Use your key in the Authorization header when calling the REST endpoints.

Managing fiscal devices

Register each physical or virtual device that will issue fiscal receipts. Devices can be associated with branches, channels, and merchants.

Issuing fiscal receipts

Send receipts to the /v1/receipts endpoint with line items, tax amounts, and payment details. FRAME Virtual Fiscalisation will sign, store, and make them available for reporting.

Configuring webhooks

Use webhooks to receive real-time notifications when receipts are accepted, rejected, or when device status changes.
